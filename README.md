# Alex Ampuero – Portfolio

Professional portfolio website showcasing backend-oriented full stack projects,
with a strong focus on e-commerce systems and product-driven development.

## Tech Stack

**Frontend**
- React · JavaScript (ES6+) · Tailwind CSS

**Tooling**
- Vite · Framer Motion · CSS Modules · Lucide React

## Technical Focus

- Responsive and accessible user interfaces
- Clear content structure and information hierarchy
- Clean, maintainable component-based architecture
- Performance-aware and SEO-friendly implementation

## Project Structure

Modular, component-based architecture focused on maintainability and scalability.

```bash
src/
├── components/
│   ├── Hero/          # Value proposition
│   ├── Navbar/        # Navigation
│   ├── Projects/      # Professional work
│   ├── Skills/        # Technical stack
│   └── Contact/       # Contact section
```

## Development

```bash
npm install
npm run dev
npm run build
npm run preview
```
